The worklog is organized as a 12-week First Cloud Journey on AWS. Each week builds on the previous one, moving from core services and networking, through databases, security, and serverless architecture, to a final capstone project.

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)

**Week 2:** [Setting up VPC networking and launching EC2 instances](1.2-week2/)

**Week 3:** [Working with IAM roles and Amazon S3 storage](1.3-week3/)

**Week 4:** [Managing databases with RDS and monitoring with CloudWatch](1.4-week4/)

**Week 5:** [Configuring DNS, AWS CLI, and database migration](1.5-week5/)

**Week 6:** [Exploring DynamoDB, Lambda, and CloudFront](1.6-week6/)

**Week 7:** [Advanced caching, VM migration, and Grafana monitoring](1.7-week7/)

**Week 8:** [Resource tagging, IAM access control, Systems Manager, and kicking off the ReviewSentinel project](1.8-week8/)

**Week 9:** [Infrastructure as Code with CDK, EC2 sizing, and cost management](1.9-week9/)

**Week 10:** [Building serverless applications with Lambda, API Gateway, and Cognito](1.10-week10/)

**Week 11:** [Designing the ReviewSentinel architecture, security model, and DynamoDB schema](1.11-week11/)

**Week 12:** [Deploying and testing the ReviewSentinel serverless application](1.12-week12/)
